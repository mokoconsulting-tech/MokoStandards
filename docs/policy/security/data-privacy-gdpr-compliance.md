## Purpose

This policy establishes requirements for protecting personal data and ensuring GDPR compliance.

## Scope

This policy applies to:
- All personal data processing activities
- EU residents' data (GDPR)
- California residents' data (CCPA)
- All systems and personnel handling personal data
