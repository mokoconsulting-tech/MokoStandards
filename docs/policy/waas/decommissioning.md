# MokoStandard: WaaS Decommissioning Policy

## Purpose

Defines mandatory procedures for securely decommissioning WaaS client sites.

## Scope

Applies to all WaaS client exits.

## Requirements

* Access revoked
* Licenses revoked or transferred
* Credentials destroyed
* Monitoring archived

## Enforcement

Mandatory for all WaaS exits.
