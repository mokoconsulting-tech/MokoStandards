# MokoStandard: WaaS Demo Baseline Policy

## Purpose

Establishes waas.demo.mokoconsulting.tech as the authoritative baseline for new WaaS provisioning.

## Scope

Applies to all new WaaS client sites.

## Baseline Rules

* All new sites must be derived from the demo baseline
* Manual builds are prohibited

## Enforcement

Validated during provisioning.
