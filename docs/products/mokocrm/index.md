# MokoCRM Documentation

## Overview

**MokoCRM** is Moko Consulting's branded Customer Relationship Management (CRM) and Enterprise Resource Planning (ERP) platform, built on Dolibarr. MokoCRM provides comprehensive business management integrating sales, customer service, project management, accounting, and more.
