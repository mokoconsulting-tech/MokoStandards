## Email Security

### Best Practices

**For sensitive communications:**
- Use PGP encryption for security@mokoconsulting.tech
- Never send credentials via email
- Use secure file sharing for sensitive documents
- Verify sender identity for financial requests

**Anti-phishing:**
- Verify @mokoconsulting.tech domain
- Check SPF/DKIM/DMARC records
- Report suspicious emails to security@mokoconsulting.tech
