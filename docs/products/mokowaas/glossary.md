# MokoWaaS Glossary

## Overview

This glossary defines terminology specific to MokoWaaS, our Joomla-based WordPress-as-a-Service platform.

## A-C

**Access Level** - Joomla permission group (Public, Registered, Special, Guest)  
**Admin Tools** - Security extension for Joomla hardening and protection  
**Akeeba Backup** - Professional backup and restoration solution for Joomla  
**Article** - Content item in Joomla (similar to WordPress post)  
**Asset** - Joomla's permission and access control object  

**Backend** - Administrator interface of Joomla (accessed via /administrator)  
**Banner** - Advertising content managed by Joomla Banners component  
**Baseline** - Standard MokoWaaS Joomla configuration  

**Cache** - Temporary storage of rendered pages for performance  
**Category** - Organizational container for Joomla content  
**CDN** - Content Delivery Network for global asset distribution  
**Component** - Major Joomla extension providing full functionality  
**Contact** - Joomla's contact management component  
**Control Panel** - Moko's custom MokoWaaS management interface  
**Core** - Joomla's base system files and functionality  

## D-F

**Database Prefix** - Table name prefix for Joomla installation  
**Demo Content** - Sample data for testing Joomla site  
**Deployment** - Process of publishing site changes to production  

**Extension** - Add-on providing additional Joomla functionality  

**Featured** - Joomla content marked for prominent display  
**Field** - Custom data element in Joomla content  
**Frontend** - Public-facing website visitors see  

## G-I

**Global Configuration** - System-wide Joomla settings  
**Group** - User permission collection in Joomla ACL  

**htaccess** - Apache configuration file for URL rewriting and security  

**Installation** - Fresh Joomla setup process  
**Instance** - Single Joomla site in MokoWaaS platform  

## J-L

**JCE Editor** - Advanced WYSIWYG editor for Joomla  
**Joomla** - Open-source CMS foundation for MokoWaaS  
**JoomlaUpdate** - Core component managing Joomla versions  

**Language** - Translation package for Joomla interface  
**Let's Encrypt** - Free SSL certificate provider  
**Library** - Shared code package used by Joomla extensions  

## M-O

**Mass Mail** - Joomla component for sending bulk emails  
**Menu** - Navigation structure in Joomla  
**Menu Item** - Individual navigation link to content  
**Menu Type** - Collection of related menu items  
**Migration** - Moving site between servers or platforms  
**MokoWaaS** - Moko Consulting's Joomla-as-a-Service platform  
**Module** - Small extension displaying content blocks  
**Module Position** - Template location where modules appear  

**Offline Mode** - Maintenance state blocking public access  
**Override** - Customized template file replacing core output  

## P-R

**Package** - Combined installer containing multiple extensions  
**Panopticon** - Multi-site monitoring and management tool  
**PerfectPublisher** - Social media integration extension  
**PHP** - Server-side programming language running Joomla  
**Plugin** - Event-triggered extension modifying Joomla behavior  
**Position** - Template location for module placement  
**Protostar** - Legacy default Joomla template  

**Redirect** - URL forwarding managed by Joomla component  
**Rollback** - Reverting to previous site state  

## S-T

**SEF URLs** - Search Engine Friendly URL rewriting  
**SEO** - Search Engine Optimization  
**Site Key** - License key for commercial Joomla extensions  
**Staging** - Testing environment separate from production  
**SSL** - Secure Sockets Layer encryption protocol  
**Super User** - Joomla's highest permission level  

**Tag** - Label for categorizing and filtering content  
**Template** - Joomla theme controlling visual appearance  
**Template Override** - Custom view file replacing core layout  

## U-W

**Update Site** - URL providing extension update information  
**Uptime** - Percentage of time site is accessible  
**User** - Individual with Joomla account and permissions  
**User Group** - Collection of users with shared access levels  

**Version Control** - Tracking content changes over time  

**Weblink** - External URL stored in Joomla Weblinks component  
**Widget** - Frontend display element (similar to module)  
**Workflow** - Content publishing process with stages  
**WYSIWYG** - What You See Is What You Get editor  

## MokoWaaS-Specific Terms

**Moko Baseline** - Standard MokoWaaS Joomla configuration with approved extensions  
**Moko Control Panel** - Custom interface for managing MokoWaaS sites  
**Moko Deployment Pipeline** - Automated workflow for site updates  
**Moko Instance** - Individual client site hosted on MokoWaaS  
**Moko Monitoring** - 24/7 uptime and performance tracking  
**Moko Snapshot** - Point-in-time backup for quick recovery  
**Moko Stack** - Optimized server configuration for Joomla  
**Moko Template** - Pre-configured Joomla setup for specific use cases  
**MokoWaaS Agent** - Software component monitoring site health  
**MokoWaaS API** - Programmatic interface for platform management  

## Platform Technologies

**Akeeba Backup Professional** - Enterprise backup solution included in MokoWaaS  
**Cassiopeia** - Modern default Joomla template (Joomla 4+)  
**Let's Encrypt Integration** - Automatic SSL certificate provisioning  
**MariaDB** - Database engine powering Joomla sites  
**Nginx** - High-performance web server in MokoWaaS stack  
**Panopticon Integration** - Multi-site dashboard and automation  
**PHP-FPM** - PHP FastCGI Process Manager for performance  
**Redis Cache** - In-memory caching for speed optimization  

## Service Terms

**Deployment Window** - Scheduled time for production changes  
**Incident Response Time** - SLA commitment for issue acknowledgment  
**Maintenance Window** - Scheduled downtime for platform updates  
**Recovery Point Objective (RPO)** - Maximum acceptable data loss timeframe  
**Recovery Time Objective (RTO)** - Maximum acceptable downtime  
**Service Level Agreement (SLA)** - Guaranteed uptime and response times  
**Uptime Guarantee** - Percentage of time site is available (e.g., 99.9%)  

## Security Terms

**Admin Tools Pro** - Security extension included in Enterprise tier  
**Brute Force Protection** - Defense against password guessing attacks  
**Firewall Rules** - Security policies blocking malicious traffic  
**Security Audit** - Regular vulnerability assessment  
**Security Hardening** - Configuration improvements for protection  
**SSL Certificate** - Digital certificate enabling HTTPS  
**Two-Factor Authentication (2FA)** - Additional login security layer  
**Web Application Firewall (WAF)** - Protection against web attacks  

## Performance Terms

**Cache Hit Ratio** - Percentage of requests served from cache  
**CDN** - Content Delivery Network distributing assets globally  
**Core Web Vitals** - Google's performance metrics (LCP, FID, CLS)  
**Database Optimization** - Query and table improvements for speed  
**Edge Caching** - CDN-level content caching  
**Gzip Compression** - File compression for faster transfers  
**Image Optimization** - Reducing image file sizes  
**Lazy Loading** - Deferring off-screen content loading  
**Page Speed Score** - Performance rating (0-100)  
**Time to First Byte (TTFB)** - Server response time metric
